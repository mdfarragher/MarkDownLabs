This module will walk you through the installation and setup of cloud services and tools that are prerequisites for this lab.

You will install the Azure Machine Learning workspace, a comprehensive platform for developing, running and evaluating machine learning pipelines in the cloud. You'll also set up a compute instance and a compute cluster for model training.
